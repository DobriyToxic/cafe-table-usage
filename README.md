# Анализ использования столов в кафе

Вариант 25 — Практика МТУСИ по СИИ

##  Описание

Веб-приложение для анализа загруженности столов в кафе с использованием предобученной нейросети YOLOv8.

##  Возможности

-  Загрузка изображений кафе
-  Автоматическая детекция столов, стульев и людей
-  Анализ занятости столов (свободен/занят)
-  Визуализация результатов с ограничивающими рамками
-  Сохранение истории всех анализов
-  Генерация отчётов в формате PDF
